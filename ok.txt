=== COMO CORRIGIR O ERRO DE BUILD NO EASYPANEL ===

O erro "undefined variable npm-9_x" acontece porque você preencheu o campo "Pacotes Nix" manualmente.

FAÇA O SEGUINTE:

1. Vá na aba "Build" do seu serviço no Easypanel.
2. No campo "PACOTES NIX", APAGUE TUDO (Deixe vazio).
3. Certifique-se que os outros campos estão assim:
   - Comando de Instalação: npm install
   - Comando de Build: echo "Pronto"
   - Comando de Início: npm start

4. Certifique-se que o arquivo "package.json" foi enviado para o seu repositório.

O Nixpacks agora vai ler o "package.json", ver que você precisa do "serve", e vai configurar o Node.js e o NPM sozinho, sem tentar adivinhar versões que não existem.

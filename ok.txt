=== CONFIGURAÇÃO EASYPANEL (NIXPACKS) ===

1. No Easypanel, aba "Build", escolha "Nixpacks".
2. Preencha os campos exatamente assim:

   - COMANDO DE INSTALAÇÃO: npm install -g serve
   - COMANDO DE BUILD: echo "Build finalizado"
   - COMANDO DE INÍCIO: serve -s . -l 80
   - PACOTES NIX: nodejs

3. Verifique se no seu repositório existem apenas estes arquivos:
   - index.html
   - main.js
   - metadata.json

=== IMPORTANTE ===
O comando 'serve -s . -l 80' garante que o site seja servido como Single Page Application e na porta 80 exigida pelo Easypanel.

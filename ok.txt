=== CONFIGURAÇÃO EASYPANEL (NIXPACKS) ===

1. No Easypanel, vá na aba "Build".
2. Selecione "Nixpacks".
3. Preencha os campos exatamente assim:

   - COMANDO DE INSTALAÇÃO: npm install -g serve
   - COMANDO DE BUILD: echo "Pronto"
   - COMANDO DE INÍCIO: serve -l 80 .
   - PACOTES NIX: nodejs

=== NOTA IMPORTANTE ===
Renomeamos 'index.tsx' para 'main.js'. O Babel no index.html ainda vai ler o arquivo como TypeScript, mas o servidor web agora aceitará o arquivo sem erros de MIME type.
